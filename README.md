# Serene Springs Community-Based Organization (CBO)

"Let's Talk, Let's Heal."

A community-driven organization committed to promoting mental wellness, empowerment, and social resilience.

## How to Run

### Prerequisites
- Node.js (v14 or higher)
- npm or pnpm

### Installation & Running Locally

1. Clone the repository:
```bash
git clone https://github.com/Kelikatulu/serene-springs-CBO.git
cd serene-springs-CBO
```

2. Install dependencies:
```bash
npm install
# or
pnpm install
```

3. Start the development server:
```bash
npm start
# or
pnpm start
```

4. Open your browser and navigate to:
```
http://localhost:3000
```

## Deployment

This project uses **GitHub Actions** for automated deployment.

**Live Site:** https://serenespringscbo.org

Every push to the `main` branch automatically builds and deploys to production.

See [DEPLOYMENT.md](DEPLOYMENT.md) for detailed setup instructions.
